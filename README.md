# MediSurveillance 🏥

A Smart Healthcare History & Disease Surveillance System designed to solve the problem of fragmented medical records across hospitals.

MediSurveillance provides a **QR-based Smart Health Card** that enables doctors to instantly access patient medical history while also helping health authorities monitor disease trends through a real-time analytics dashboard.

## 🚨 Problem Statement

Medical records are often **scattered across multiple hospitals and clinics**, making it difficult for doctors to quickly access a patient's complete medical history.

This lack of centralized information can lead to:

- Delayed diagnosis
- Repeated medical tests
- Poor healthcare coordination
- Limited disease trend monitoring

---

## 💡 Our Solution

We built **MediSurveillance**, a healthcare platform that:

- Stores and manages patient medical records
- Uses a **QR-based Smart Health Card** for instant patient identification
- Allows doctors to quickly access patient information
- Provides **disease trend analytics dashboards**
- Enables efficient healthcare monitoring

# 👥 Role-Based System

The platform supports **three types of users**:

## Patient Portal

Patients can access a personal healthcare dashboard.

Features include:

- Overview Dashboard
  - Active Cases
  - Under Treatment
  - Recovered Cases
  - Disease Alerts

- Disease Analytics
  - Six-Month Disease Trends
  - Disease Distribution Charts

- Health Records
  - View complete medical history

- Appointment Management
  - View confirmed appointments
  - Track appointment schedules

- Medicine Reminders
  - Notifications for prescribed medicines

- QR Scanner
  - Scan smart health card for quick identification

---

## Doctor Portal

Doctors can monitor patients and manage healthcare data.

Features include:

- Dashboard Overview
  - Total Patients
  - Active Cases
  - Recovered Patients
  - Disease Alerts

- Disease Analytics
  - Six-Month Disease Trend
  - Disease Distribution

- Patient Management
  - View all assigned patients
  - Register new patients

- Appointment Management
  - Schedule appointments
  - Reschedule appointments
  - Cancel appointments

- QR Scanner
  - Scan patient health cards to instantly retrieve medical details

---

## Admin Portal

Admins manage the entire healthcare system.

Features include:

- System Dashboard
  - Total Patients
  - Active Cases
  - Recovered Patients
  - Disease Alerts

- Registration Analytics
  - Six-Month Patient Registrations
  - Disease Resolution Tracking

- Patient Management
  - View all registered patients
  - Register new patients

- Appointment Monitoring

- Alert Management

---

# 📱 QR-Based Smart Health Card

Each patient receives a **QR-based health card**.

Doctors can scan the QR code to instantly access patient details:

- Patient Name
- Address
- Age
- Current Disease
- Assigned Doctor
- Medical History

This allows **quick patient identification during medical emergencies**.

## ⚙️ How to Run the Project

## Requirements

- Node.js installed
- npm installed
- Modern browser (Chrome, Edge, Firefox)

Follow these steps to run the project locally.

### 1. Clone the repository
```bash
git clone https://github.com/sanchita23/Smart-Healthcare-Hackathon.git
```

### 2. Navigate into the project folder
```bash
cd Smart-Healthcare-Hackathon
```

### 3. Install dependencies
```bash
npm install
```

### 4. Start the development server
```bash
npm run dev
```

### 5. Open in browser
Visit:

```
http://localhost:5173
```
